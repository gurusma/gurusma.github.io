16. Soal simulasi UN 6log72-6log4+6log24-6log2 adalah ...


^6log72-^6log4+^6log24-^6log2
= ^6log(72/4 x 24/2)
= ^6log(36/1 x 6/1)
= ^6log(36x6)
= ^6log6³
= 3.(^6log6)
=3(1)=36Log(72/4 x 24/2) = 6Log216
6Log216 = 3
