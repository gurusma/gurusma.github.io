Mapel: Matematika

Kelas: 1 SMP

Materi: Aritmatika

Kata Kunci: ‘width’ ‘circumference’ ‘length’

Kode Soal: 2

Kode Kategorisasi: 7.2

Jawaban:

A rectangle has a length (2x + 3) cm and a width (x + 8) cm, If the circumference of the rectangle is 64 cm, the width of the rectangle is 15 cm

Solution :

• x

Circumference = 2 × (length + width)

64 cm = 2 × (2x + 3 + x + 8)

64 cm = 2 × (3x + 11)

64 cm = 6x + 22

6x = 42

x = 7

• Width

Width = x + 8

Width = 7 + 8

Width = 15 cm
