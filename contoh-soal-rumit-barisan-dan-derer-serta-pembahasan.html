38. tolong jelasin materi persamaan kuadrat!mulai dari awal sampai akhir. (penjelasan jgn bertele tele dan kasih soal RUMIT serta pembahasan).​


Pembahasan

persamaan kuadrat adalah persamaan yang memiliki fungsi kuadrat seperti x² nah bentui formalnya adalah :

ax² + bx + c, a ≠ 0

lah kenapa harus ada a ≠ 0 ?? Nah itulah, itu karena persamaan kuadrat harus ada fungsi kuadrat, begini, kamu mau belajar fungsi kyadrat pasti udah belajar fungsi linear kan??? misalkan kalau persamaan kuadrat dengan a = 0 :

(0) x² + (3) x + 6 = 0

3x + 6 = 0

nah, malah jadi fungsi linear, karena tidak ada koefisien yang bersama variabel x². Oke, kita langsung masuk materi pertama, yaitu mencari akar akar, 2 metode yaitu :

Pemfaktoran

ada 4 metode pemfaktoran, yaitu :

Metode 1 : untuk a = 1

Jika persamaan kuadrat : x² + bx + c = 0

maka faktor faktornya : (x + p) (x + q) = 0

dengan syarat yaitu : p + q = b, pq = c

maka akar akarnya (x1 dan x2) : -p dan -q

Metode 2 : untuk x ≠ 0

Jika persamaan kuadrat : ax² + bx + c = 0

maka faktor faktornya : (dx + e)² = 0

dengan syarat yaitu : e² = c, 2dex = bx, d² = a

maka akar akarnya : x1 = x2 = -e/d

Metode 3 : untuk b = 0, c < 0, a ≠ 0

Jika persamaan kuadrat : x² + bx + c = 0

maka faktor faktornya : (x + √c) (x -√c) = 0

maka akar akarnya (x1 dan x2) : √c dan -√c

Metode 4 : untuk c = 0, a ≠ 0

Jika persamaan kuadrat : x² + bx + c = 0

maka faktor faktornya : x (ax + b) = 0

maka akar akarnya (x1 dan x2) : 0 dan -b/a

Rumus ABC

Oke, sebelum tahu rumusnya, saya mau kamu tahu asal usul rumusnya, rumus ABC ini digunakan untuk mencari nilai akar akar nya. Oke kita mulai dari bentuk formal.

ax² + bx + c = 0

[tex] {x}^{2} + \frac{b}{a} x + \frac{c}{a} = 0[/tex]

[tex] {x}^{2} + \frac{b}{a} x + { (\frac{b}{2a} )}^{2} = - \frac{c}{a} x + { (\frac{b}{2a} )}^{2}[/tex]

[tex] {x}^{2} + \frac{b}{a} x + \frac{ {b}^{2} }{4 {a}^{2} } = - \frac{c}{a} x + \frac{ {b}^{2} }{4 {a}^{2} } [/tex]

Nah kita bisa menggunakan metode pemfaktoran kedua, yaitu

Jika persamaan kuadrat : ax² + bx + c = 0

maka faktor faktornya : (dx + e)² = 0

dengan syarat yaitu : e² = c, 2dex = bx, d² = a

Maka faktornya adalah :

[tex]{(x + \frac{b}{2a} ) }^{2} = - \frac{4ac}{4 {a}^{2} } x + \frac{ {b}^{2} }{4 {a}^{2} } [/tex]

namun apakah syaratnya mengikuti? Oke kita tes, pertama :

e² = c

[tex] {( \frac{b}{2a} })^{2} = \frac{ {b}^{2} }{4 {a}^{2} } [/tex]

[tex] { \frac{ {b}^{2} }{4 {a}^{2} } } = \frac{ {b}^{2} }{4 {a}^{2} } \: (Benar)[/tex]

kedua :

2dex = bx

[tex]2(1)( \frac{b}{2a} )x = \frac{b}{a}x [/tex]

[tex] \frac{b}{a}x= \frac{b}{a}x \: (Benar)[/tex]

ketiga :

d² = a

1² = 1

1 = 1 (Benar)

maka faktorisasinya benar, lanjutkan rumusnya

[tex]{(x + \frac{b}{2a} ) }^{2} = \frac{ {b}^{2} - 4ac}{4 {a}^{2} } [/tex]

[tex]{x + \frac{b}{2a}} = \sqrt{ \frac{ {b}^{2} - 4ac}{4 {a}^{2} } }[/tex]

[tex]{x + \frac{b}{2a}} = ±\frac{ {b}^{2} - 4ac}{2a } [/tex]

[tex]x = \frac{ - b ± \sqrt{ {b}^{2} - 4ac} }{2a} [/tex]

maka itulah rumus ABC

Rumus yang lain dari ax² + bx + c = 0, a ≠ 0 adalah :

x1 + x2 = -b/a

x1 × x2 = c/a

Namun, ada 1 hal lagi yaitu “Diskriminan.” Aku yakin kamu tidak mengerti apa itu “akar akar” dam fungsi akar akar itu sendiri, ya itu akan kamu pelajari du materi berikutnya yaitu “fungsi kuadrat.” Namun diskriminan salah satu fungsinya adalah untuk menentukan apakah dia mempunyai akar akar atau tidak. Rumus diskriminan adalah :

D = b² -4ac

D adalah simbol diskriminan, dengan ini, kita bisa menyederhanakan rumus ABC. Yaitu dari :

[tex]x = \frac{ - b ± \sqrt{ {b}^{2} - 4ac} }{2a} [/tex]

menjadi

[tex]x = \frac{ - b ± \sqrt{ D} }{2a} [/tex]

nah itu, untuk jenis diskriminan sendiri, jenis nya ada 3 yaitu :

D < 0, Jika diskriminannya kurang dari 0, maka dalam persamaan kuadrat persamaan itu tidak bisa difaktorkan, namun dalam funvlgsi kuadrat, persamaan itu tidak menyinggung sumbu y.D = 0, Jika diskriminannya sama dengan 0, maka dalam persamaan kuadrat metode pemfaktorannya bisa menggunakan metode kedua dari metode pemfaktoran yang sudah saya bahas, tapi dalam fungsi kuadrat, persamaan itu menyinggung sumbu y di satu titik.D > 0, Jika diskriminannya lebih dari 0, maka dalam persamaan kuadrat metode pemfaktorannya tidak bisa menggunakan metode kedua, tapi dalam fungsi kuadrat, persamaan itu memotong sumbu y di dua titik.

Contoh Soal

Diketahui salah satu akar dari persamaan kuadrat x² + 3x + c = 0 adalah 4. Tentukan nilai akar lainnya!

Penyelesaian

diketahui akar persamaan kuadrat

x² + 3x + c = 0

x1 = 4

x2 = ??

maka tentukan nilai c dulu :

x² + 3x + c = 0

(4)² + 3(4) + c = 0

16 + 12 + c = 0

28 + c = 0

c = -28

maka persamaan kuadratnya adalah :

x² + 3x -28 = 0

Setelah itu, kita bisa memfakforkan dengan metode pertama pemfaktoran karena a = 1.

persamaan kuadrat : x² + 3x -28 = 0

maka syarat membentuk faktornya adalah :

p + q = b = 3

p × q = c = -28

nilai p × q yang memenuhi adalah

-28 × 1 = -28

-14 × 2 = -28

-7 × 4 = -28

-4 × 7 = -28

-2 × 14 = -28

-1 × 28 = -28

faktor -28 yang jumlahnya adalah b atau 3 yaitu -4 × 7 karena p + q = b dimana

-4 + 7 = 3 (Benar). Maka :

p = -4

q = 7

namun akar akarnya adalah -p dan -q yang dimana itu adalah 4 dan -7. Maka akar lainnya adalah -7.
