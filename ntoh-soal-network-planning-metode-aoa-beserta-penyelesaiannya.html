Apa saja manfaat network planning?

- Perencanaan suatu proyek yang kompleks.  

- Scheduling pekerjaan-pekerjaan sedemikian rupa dalam urutan yang praktis dan efisien.

4. Selesaikan persamaan berikut dan beraoa nilai x |7-2x|-11=14


Jawaban:

x = -9 atau x = 16

Penjelasan dengan langkah-langkah:

|7 - 2x| - 11 = 14

|7 - 2x| = 14 + 11

|7 - 2x| = 25

jika 7 - 2x positif, maka

7 - 2x = 25

2x = 7 - 25

2x = -18

x = -9

Jika 7 - 2x negatif, maka

-(7-2x) = 25

2x - 7 = 25

2x = 25 + 7

2x = 32

x = 16

Jadi, nilai untuk x adalah x = -9 atau x = 16

Network layer
Berfungsi untuk mendefinisikan alamat-alamat IP, membuat header untuk paket-paket, dan kemudian melakukan routing melalui internetworking dengan menggunakan router dan switch layer-3.

6. Fungsi network layer beserta penjelasannya


-network layer

Πfungsi network layer :
1.)menentukan tujuan data pada jaringan
Ketika sebuah data akan ditransmisikan melalui sebuah jaringna komputer, maka pastinya, paket data tersbut memiliki tujuan.
Misalnya adalah dari server, menuju klien. 

2.)mendefinisikan alamat IP
address merupakan sistem pengalamatan bagi jaringan komputer yang menggunakan protocol TCP/IP dimana setiap komputer atau lebih tepatnya network interface card  atau NIC memilki alamat IP tersendiri, sesuai dengan host yang digunakan. 

3.)membuat header pada pekat²data yang ada
Sebuah data bernama EFGHI. EFGHI dalam proses transmisi data akan dipecah ke dalam paket data. Tugas atau fungsi dari network layer ini adalah memberikan header pada EFGHI, dengan nama EFGHI 3, EFGHI 4, EFGHI 5, dan seterusnya, sesuai dengan jumlah banyaknya pecahan data yang ditransmisikan melalui jaringan komputer.

4.)melakukan proses routhing
network layer mendistribusikan atau menghala setiap paket data yang masuk ke dalam jaringan komputer menuju lokasi masing-masing, sesuai dengan IP address yang sudah didefinisikan sebelumnya. Network layer akan memberikan ‘jalan’ atau rute bagi setiap paket data yang akan ditransmisikan untuk mencapai komputer tujuannya.

18. cara kerja network layer beserta penjelasan


Network Layer
Salah satu layer atau lapisan yang terdapat di dalam sistem lapisan OSI layer yang cukup memiliki peran penting adalah network layer. Lapisan yang bekerja pada tingkat ke 5 ini merupakan lapisan yang memiliki peran penting dalam proses transmisi jaringan komputer.

Network layer, yang merupakan lapisan ke lima pada keseluruhan sistem jaringan OSI Layer memiliki beberapa fungsi dalam jarignan komputer. Berikut ini adalah beberapa fungsi dari network layer :

1)Menentukan tujuan data pada sebuah jaringan
2) Mendefinisikan alamat IP
3) Membuat header pada paket – paket data yang ada
4) Melakukan proses routing

✯-----------------------------------------✯

☞︎︎︎JAWABAN

Campuran homogen adalah larutan. Dimana , larutan merupakan campuran dari zat pelarut dan terlarut. Campuran heterogen adalah campuran yang tidak dapat dicampur hingga menjadi suatu larutan.

Contoh Homogen

Contoh campuran homogen adalah :

- Air Garam

- Air Gula

Contoh Heterogen

Contoh campuran heterogen adalah:

-Air dan minyak

-Air dan pasir

☞︎︎︎PENJELASAN LEBIH LENGKAP/CARA

Ciri-ciri Homogen

Salah satu dari Ciri-ciri campuran homogen adalah:

-Campurannya tidak akan mengendap/akan tercampur meskipun tetap dibiarkan

Ciri-ciri Heterogen

Ciri-ciri campuran heterogen adalah:

-Campurannya akan mengendap/membentuk ampas jika dibiarkan

30. tuliskan 2 komponen Network beserta penjelasannya


End Device

Perangkat jaringan yang paling familiar bagi kita sehari-hari yaitu end devices atau hosts. Perangkat ini memberi bentuk bagi antarmuka/interface antara pengguna dan jaringan komunikasi dasar. Beberapa contoh end device yaitu:

Komputer (work station, laptop, file server, web server)

Printer jaringan

Telepon VoIP

TelePresence endpoint

Kamera pengawas

Perangkat genggam bergerak (smartphone, tablet, PDA dan wireless debit/credit card reader dan barcode scanner)

Sebuah perangkat host, baik yang menjadi sumber atau tujuan dari sebuah pesan, dikirimkan melalui suatu jaringan. Supaya dapat membedakan antara satu host dengan host lainnya, setiap host diidentifikasikan dengan/diberikan sebuah alamat/address. Pada saat sebuah host memulai komunikasi, host tersebut menggunakan alamat dari host tujuan untuk menentukan ke mana tujuan pesan tersebut.

Intermediary Device

Perangkat intermediary berfungsi untuk menghubungkan antar end device. Perangkat ini menyediakan sambungan atau konektivitas dan bekerja untuk menjamin aliran data yang melalui jaringan. Selain menghubungkan host secara individu ke jaringan, perangkat intermediary juga dapat menghubungkan beberapa jaringan individu untuk membentuk sebuah internetwork. Contoh perangkat intermediary yaitu:

Network Access (switch dan wireless access point)

Internetworking (router)

Security (firewall)

Mengatur aliran data melalui jaringan juga merupakan fungsi dari perangkat intermediary. Dalam hubungannya dengan jaringan interkoneksi, perangkat-perangkat ini juga menggunakan alamat host tujuan untuk menentukan jalur yang mesti ditempuh oleh pesan.

Proses-proses yang berjalan dalam perangkat jaringan intermediary, menampilkan fungsi-fungsi sebagai berikut:

Membangkitkan dan mengirimkan lagi sinyal-sinyal data

Mengurusi informasi yaitu jalur mana yang tersedia di jaringan dan internetwork

Memberi notifikasi kepada perangkat lain akan adanya eror dan kegagalan komunikasi

Mengarahkan data untuk melalui jalur alternatif pada saat terjadi kegagalan sebuah sambungan

Mengklasifikasi dan mengarahkan pesan berdasarkan prioritas Quality of Service (QoS)

Menyetujui atau tidaknya sebuah aliran data berdasarkan pengaturan keamanannya.

38. mengatasi routing loop dengan metode mengubah network yang putus menjadi penambah satu dari maksimum loop adalah metode yang bernama?​


Metode yang digunakan untuk mengatasi routing loop dengan mengubah network yang putus menjadi penambah satu dari maksimum loop adalah metode "Route Splitting" atau "Split Horizon".

Dalam routing loop, terjadi situasi di mana paket data terus-menerus mengikuti jalur yang berulang-ulang antara dua atau lebih node jaringan. Hal ini dapat terjadi karena informasi rute yang tidak akurat atau adanya konfigurasi yang salah pada protokol routing. Salah satu metode yang digunakan untuk mengatasi masalah ini adalah dengan menerapkan Split Horizon.

Pada metode Split Horizon, setiap router atau node dalam jaringan akan menghindari mengirimkan informasi rute kembali ke arah asal dari mana informasi itu diterima. Dengan kata lain, setiap node akan menahan atau mengabaikan pengiriman informasi rute kembali ke jalan yang sama yang telah ditempuh.

Dengan menerapkan Split Horizon, network yang putus menjadi penambah satu dari maksimum loop, karena node yang menerima informasi rute tidak akan mengirimkannya kembali ke arah asal, sehingga menghindari terjadinya routing loop yang berkelanjutan.

Metode Split Horizon ini merupakan salah satu teknik yang digunakan dalam algoritma routing untuk mengatasi routing loop dan memastikan efisiensi dan kestabilan jaringan yang lebih baik.

40. Buatkan 4 paragraf karangan tentang planning Kita setelah lulus SMK beserta artinya


Karangan tentang planning setelah lulus SMK:

I have a plan to continue my studies in computer science in a reputable university here in Bandung. I want to be a computer scientist, developing many applications and softwares for students.

And after completing my studies in the university i will gain work experience in many reputable companies around the world. I will develop my career to the top level. I will move up the career path in the company until I have enough experience to start my own company.

That is my first planning. The second is I will finish my studies at university with high GPA and then start working freelance in various software companies so that I have a lot of experience.

And with an abundance of valuable experience I will start developing my own software. And of course I will keep working as a freelancer for my time and financial freedom.

Pembahasan

Terjemahan teks di atas adalah:

Saya memiliki rencana untuk melanjutkan studi saya di bidang ilmu komputer di salah satu universitas ternama di Bandung. Saya ingin menjadi ilmuwan komputer, mengembangkan banyak aplikasi dan perangkat lunak untuk siswa.

Dan setelah menyelesaikan studi saya di universitas saya akan mendapatkan pengalaman kerja di banyak perusahaan terkemuka di seluruh dunia. Saya akan mengembangkan karir saya ke level tertinggi. Saya akan menapaki jenjang karir di perusahaan sampai saya memiliki pengalaman yang cukup untuk memulai perusahaan saya sendiri.

Itu adalah perencanaan pertama saya. Yang kedua adalah saya akan menyelesaikan studi saya di universitas dengan IPK tinggi dan kemudian mulai bekerja freelance di berbagai perusahaan perangkat lunak sehingga saya memiliki banyak pengalaman.

Dan dengan banyak pengalaman berharga tersebut saya akan mulai mengembangkan perangkat lunak saya sendiri. Dan tentu saja saya akan tetap bekerja sebagai freelancer untuk waktu dan kebebasan finansial.

Menceritakan tentang planning pribadi berarti membicarakan sesuatu yang akan kita lakukan di masa depan. Hal tersebut berarti berbicara dengan future tense. Maka kita bisa memakai format will + verb, atau be + going to.
