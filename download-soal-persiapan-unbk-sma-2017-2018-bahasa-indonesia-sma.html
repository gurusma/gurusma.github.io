5. Soal Latihan USBN 2017 / 2018 Mapel: Matematika (Peminatan) Satuan Pendidikan: SMA Tahun: 2017 / 2018 Kurikulum: Kurikulum 2013


Kelas : 11
Mapel : Matematika
Kategori : Trigonometri Lanjut, Limit
Kata Kunci : Sudut rangkap, persamaan trigonometri, Limit x mendekati tak terhingga
Kode : 11.2.3 (Kelas 11 Matematika Bab 3 - Trigonometri Lanjut), 11.2.7 (Kelas 11 Matematika Bab 7 - Limit)

Pembahasan :

1) cos 2x - 3 cos x + 2 = 0
=> (2 cos² x - 1) - 3 cos x + 2 = 0
Misal cos x = a
=> (2a² - 1) - 3a + 2 = 0
=> 2a² - 3a + 1 = 0
=> (2a - 1)(a - 1) = 0
=> a = 1/2 atau a = 1

Untuk a = 1/2
cos x = 1/2
Kuadran 1 : x = 60° = π/3
Kuadran 4 : x = 300° = (5/3) π

Untuk a = 1
cos x = 1
x = 0° = 0
x = 360° = 2π

HP = {0, π/3, (5/3) π, 2π}


2) Rumus yang digunakan :
Lim √(ax² + bx + c) - √(px² + qx + r)
x=>~
Jika a > p => ~
Jika a = p => (b - q)/(2√a)
Jika a < p => -~

Lim √(4x² + 6x - 7) - 2x - 3
x=>~

Lim √(4x² + 6x - 7) - (2x + 3)
x=>~

Lim √(4x² + 6x - 7) - √(2x + 3)²
x=>~

Lim √(4x² + 6x - 7) - √(4x² + 12x + 9)
x=>~

a = 4, b = 6, c = -7, p = 4, q = 12, r = 9
Karena a = p maka bisa menggunakan rumus :

(b - q)/(2√a)
= (6 - 12)/(2√4)
= -6/4
= -3/2
= -1 1/2
= -1,5

12. Mosi unbk harus dilaksanakan di seluruh sma/smk


Mosi: UNBK (Ujian Nasional Berbasis Komputer) harus dilaksanakan di seluruh SMA/ SMK

Pendapat Pro:

Setuju, sebab bila UNBK dilaksanakan maka seluruh SMA/ SMK dapat menghemat kertas. Hal ini dapat pula memangkas biaya kegiatan Ujian Nasional (dapat meringankan beban orang tua murid).

Pendapat Kontra:

Tidak setuju, sebab tidak semua SMA/ SMK memiliki fasilitas komputer yang layak. Untuk menunjang kelancaran kegiatan UNBK sudah seharusnya memakai komputer dengan spesifikasi yang memadai. Sedangkan seluruh SMA/ SMK belum tentu memilikinya (dapat membebani pihak sekolah, bila UNBK dilaksanakan).

33. contoh kalimat penutup pidato persuasif bertema persiapan UNBK 2018


Contoh kalimat penutup pidato persuasif bertema persiapan UNBK 2018 sebagai berikut :

Demikian kiranya yang dapat saya sampaikan, Terimakasih atas perhatian dan waktunya sudah berkenan mendengarkan pidato saya. Mohon maaf bila ada salah kata yang terucap. Semoga apa yang telah saya sampaikan dapat bermafaat bagi kita semua.
Akhirnya saya ingin menutup pidato ini dengan sebuah kalimat motifasi "KAMU TIDAK HARUS MENJADI HEBAT UNTUK MEMULAI, TAPI KAMU HARUS MEMULAI UNTUK MENJADI HEBAT" Kiranya boleh menjadi penyemangat kita memulai UNBK 2018. Akhir kata, saya ucapkan wasaalamu’alaikum warahmatullahi wabakatuh.

PEMBAHASAN
PIDATO PERSUASIF adalah : Pesan yang disampaikan pembicara untuk mempengaruhi, mengajak, membujuk pendengar untuk mengikuti keinginan pembicara secara sukarela.

Untuk dapat membuat sebuah pidato, kita harus mengetahui apa saja struktur dari sebuah pidato.

1. PEMBUKAAN PIDATO ada 3 bagian yaitu: salam pembuka, ucapan penghormatan, hingga ucapan syukur.
