3. Rumus Simpe future, Simple perrect, past tense , simpe present tense apa ya? TOLONG DIJAWAB!!!


- simple future tense : s + will be + v1
- simple perfect tense :
kalo past perfect : s + had been + v3
kalo present perfect : s + has/have been + v-ing
kalo future perfect : s + will have been + v3
- past tense : s + was/were + v1 atau S + v2
- simple present : s + is/am/are + v-ingJawaban :
--Tenses--
Simple Future Tense :
Nominal :
S + will be + V/Complement
Verbal :
S + will + V1

Perfect Tense :
Verbal :
(+) S + has/have + V3
(-) S + has/have not + V3
(?) Has/have + S + V3 + ?
Nominal :
(+) S + has/have been + Complement

Simple Past Tense :
Nominal :
She + tobe (was/were) + complement
Verbal :
(+) S + V2 + Complement
(-) S + did not + V2
(?) Did + S + V1

Simple Present Tense :
Nominal :
(+) S + tobe (am is are) + complement
(-) S + tobe + not + complement
(?) Tobe + S + complement + ?
Verbal :
(+) S + V1 (s/es)
(-) S + do/does not + V1
(?) Do/does + S + V1 + ?

1. The man eats some mangoes.

2. The man was eating some mangoes.

3. The man has eaten some mangoes.

Penjelasan:

Jawaban:

Simple present tense: The man eats some mangoes

Present continuous tense: The man is eating some mangoes

Present perfect tense: The man has eaten some mangoes

Penjelasan:

Kalimat di atas sudah termasuk ke dalam simple present tense (subject + verb with s/es)

Rumus dari present continuous tense adalah subject + am/is/are + verb -ing

Rumus dari present perfect tense adalah subject + has/have + verb 3.

The man pada kalimat di atas merupakan orang ketiga tunggal, sehingga menggunakan is atau has.
