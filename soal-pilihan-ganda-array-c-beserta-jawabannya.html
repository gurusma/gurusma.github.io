Hasil dari [tex]\begin{bmatrix}-1 &3 \\ 2&-1 \\\end{bmatrix}\times\begin{bmatrix}-3 &-1 \\0 &2 \\\end{bmatrix}[/tex] adalah [tex]\begin{bmatrix}3 &7 \\-6 &-4 \\\end{bmatrix}[/tex].

PEMBAHASAN

Matriks merupakan suatu susunan bilangan yang disusun dalam m baris dan n kolom sehingga membentuk suatu susunan persegi. Matriks m x n berarti matriks tersebut memiliki m baris dan n kolom.

Salah satu operasi pada matriks adalah perkalian. Suatu matriks A dapat dikalikan dengan matriks B apabila jumlah kolom pada matriks A sama dengan jumlah baris pada matriks B. Sehingga matriks [tex]A_{m\times n}[/tex] dapat dikalikan dengan matriks [tex]B_{p\times l}[/tex] jika n = p. Hasil perkaliannya adalah matriks dengan baris m dan kolom l atau matriks [tex]C_{m\times l}[/tex] .

Contoh perkalian matriks 2x2 :

[tex]\begin{bmatrix}a &b \\c &d \\\end{bmatrix}\times\begin{bmatrix}p &q \\r &s \\\end{bmatrix}=\begin{bmatrix}ap+br &aq+bs \\cp+dr &cq+ds \\\end{bmatrix}[/tex]

Contoh perkalian matriks 3x2 dan matriks 2x1 :

[tex]\begin{bmatrix}a &b \\c &d \\ e&f \\\end{bmatrix}\times\begin{bmatrix}p \\q\end{bmatrix}=\begin{bmatrix}ap+bq \\ cp+dq\\ep+fq\end{bmatrix}[/tex]

Penjelasan:

Berikut adalah codingan untuk mengshuffle isi array dalam bahasa C

#define _CRT_SECURE_NO_WARNINGS

#include<stdio.h>

#include<stdlib.h>

#include<time.h>

void printArray(int array[], int n)

{

for(int i=0; i<n; i++)

{

 printf("%d ",array[i]);

}

printf("\n");

}

void shuffleArray(int array[], int n)

{

int temp, index;

for(int i=0; i<n; i++)

{

 index = rand()%n;

 temp = array[index];

 array[index] = array[i];

 array[i] = temp;

}

}

int main()

{

int n;

int array[] = {12,14,15,23,25,27};

n = 6;

srand(time(NULL));

printf("List Array Before:\n");

printArray(array,n);

shuffleArray(array,n);

printf("List Array After:\n");

printArray(array,n);

getchar();

return 0;

}

Lalu dishuffle dengan lakukan iterasi sebanyak ukuran array dan tiap iterasi pilih index array secara acak dengan menggunakan function rand() yang dibatasi seukuran isi array (Jika rand()%10 berarti random antara 0-10) dan lakukan swap dengan array ke iterasi itu.
