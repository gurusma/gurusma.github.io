1. penjelasan tentang pemograman web,pemograman dasar, dan sistem operasi


pemograman web :
Pemrograman yang dalam bahasa English adalah programming dan diartikan proses, cara, perbuatan program. Definisi Web : jaringan komputer yang terdiri dari kumpulan situs internet yang menawarkan teks dan grafik dan suara dan sumber daya animasi melalui protokol transfer hypertext.

pemograman dasar ;
Pemrograman dasar adalah proses menulis, menguji dan memperbaiki (debug), dan memelihara kode yang membangun sebuah program komputer. Kode ini ditulis dalam berbagai bahasa pemrograman.
sistem operasi :
istem Operasi (Operating System : OS) adalah komponen pengolah peranti lunak dasar (essential component) tersistem sebagai pengelola sumber daya perangkat keras komputer (hardware), dan menyediakan layanan umum untuk aplikasi perangkat lunak.

14. jelaskan uraian tipe data, variable, parameter dalam implementasi pemograman web​


Tipe data adalah jenis dari suatu nilai yang dapat diberikan pada sebuah variabel atau parameter. Contoh tipe data di pemrograman web adalah integer, string, boolean, dan lain-lain.

Variabel adalah tempat penyimpanan sementara suatu nilai pada saat proses pemrograman. Variabel memiliki nama dan tipe data yang dapat diubah-ubah sesuai dengan kebutuhan. Variabel digunakan untuk menyimpan nilai yang akan digunakan dalam proses pemrograman.

Parameter adalah suatu nilai yang dikirim ke dalam sebuah fungsi atau prosedur. Parameter dapat digunakan untuk mengirimkan informasi ke dalam sebuah fungsi atau prosedur yang dapat digunakan untuk melakukan tugas tertentu.

Dalam implementasi pemrograman web, tipe data, variabel, dan parameter sering digunakan untuk menyimpan dan mengolah data yang dibutuhkan dalam proses pemrograman. Mereka juga dapat digunakan untuk mengirim dan menerima informasi dari fungsi atau prosedur yang dijalankan dalam sebuah aplikasi web.

19. sebutkan 5 bahasa pemograman beserta nama penemunya​


Jawaban:

JAVA. Logo Java. Java merupakan bahasa pemrograman yang dapat dijalankan di berbagai platform, dari mulai PC sampai perangkat mobile. ...2. C. Bahasa C. ...PHP. Logo PHP. ...4. Visual Basic. Logo Visual Basic. ...Python. Logo Python. ...6. C++ C++ ...7. JavaScript. JavaScript. ...8. C# C# adalah bahasa pemrograman yang dikembangkan oleh Microsoft.

1. Java

Java adalah salah satu bahasa khusus untuk pemrograman yang dijalankan pada berbagai macam jenis platform, seperti perangkat mobile hingga perangkat komputer. Pada mulanya bahasa pemrograman ini dikembangkan pada tahun 1991 dari The Green Project Sun Microsystem. Kabarnya, nama Java diambil oleh sang penemu yakni James Gosling yang sangat menyukai kopi tubruk murni dari Jawa, Indonesia.

Sebagai bahasa pemrograman yang sangat populer di dunia, Java mempunyai kelebihan seperti mampu berjalan di OS berbeda serta termasuk ke dalam kategori pemrograman orientasi objek. Di samping itu, bahasa pemrograman ini juga memiliki library sendiri sehingga mampu dengan cepat memudahkan programer.

2. C

Jika Java adalah bahasa pemrograman terpopuler, bahasa C merupakan bahasa pemrograman tertua. Bahasa ini dikembangkan sudah sejak tahun 70an dan dibuat oleh Dennis Ritchie. Bahasa C dianggap sebagai bapak dari bahasa pemrograman lain tidak lain karena menjadi dasar untuk mempelajari lebih lanjut bahasa lainnya seperti Java, C++, C# dan lainnya.

Pada zamannya, bahasa C sangat populer layaknya Java karena digunakan hampir di sebagian besar perguruan tinggi jurusan IT. Singkatnya, seseorang tidak bisa belajar bahasa pemrograman lain sebelum belajar bahasa C.

3. PHP

PHP atau yang familiar dengan hypertext preprocessor adalah semacam bahasa pemrograman paling umum digunakan di Indonesia. Tidak hanya gratis, PHP umum dipakai karena berisikan server side scripting guna membangun web yang dinamis. Tidak heran jika Anda banyak menjumpai istilah ini di berbagai buku pemrograman.

Di samping itu, ada begitu banyak web populer yang juga menggunakan bahasa pemrograman yang dikembangkan oleh Rasmus Lerdorf. Misalnya, Facebook, Digg dan WordPress.

4. Visual Basic

Visual Basic adalah semacam bahasa pemrograman yang dikembangkan oleh developer ternama dunia yakni Microsoft. Bahasa khusus pemrograman ini menawarkan ide visual guna pembuatan software khusus berbasis sistem operasi Windows. Bahasa pemrograman ini sendiri merupakan turunan dari bahasa pemrograman BASIC.

Saat ini, Visual Basic juga termasuk ke dalam kategori bahasa pemrograman populer Indonesia. Bahkan sekarang sudah terdapat grup Visual Basic Indonesia dengan anggota mencapai 22 ribu lebih.

5. Python

Apabila dilihat berdasarkan tingkatan bahasa pemrograman, Python termasuk ke dalam kategori bahasa tingkat tinggi. Python mampu membangun aplikasi baik itu dengan basis web hingga mobile sekalipun. Meskipun terbilang bahasa tingkat tinggi, akan tetapi python juga sangat mudah dipahami bagi pemula.

Hal ini dikarenakan bahasa tersebut sangat gampang dibaca, termasuk untuk memahami syntaxnya. Hingga sekarang terdapat begitu banyak perusahaan besar yang memakai bahasa Python untuk pengembangannya seperti Yahoo, Google dan NASA.

32. sebutkan simbol simbol pemograman flowchart beserta fungsinya..!


Sebutkan simbol simbol pemograman flowchart beserta fungsinya, penggunaan flowchart dapat digunakan untuk menggambarkan bagaimana suatu proses terjadi. Flowchart digambarkan dengan beberapa simbol. Simbol-simbol yang sering digunakan pada flowchart adalah 

1. Tanda panah, tanda panah dapat digunakan sebagai penghubung antara simbol yang satu dengan yang lainnya, tanda panah pada flowchart berarti suatu proses yang berjalan.

2. Lonjong, pada simbol yang kedua adalah persegi panjang namun pada kedua lebarnya berbentuk setengah lingkaran, simbol ini berarti awal dari suatu proses. Selain sebagai awal suatu proses, simbol ini juga bisa digunakan untuk akhir suatu proses.

3. Lingkaran, simbol flowchart yang ketiga adalah lingkaran. Simbol lingkaran berarti suatu proses yang berjalan keluar masuk pada menu atau halaman yang sama.
