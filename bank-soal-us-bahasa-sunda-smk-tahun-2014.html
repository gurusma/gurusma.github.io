30. pada tahun 2014 data keuangan Filipina mencatat pendapatan domestik bruto PDB riil sebesar 272,02 miliar US Dollar pada tahun 2013 PDB Filipina sebesar 250,18 miliar US Dollar Berdasarkan data tersebut hitunglah laju pertumbuhan ekonomi Filipina tahun 2014


Diketahui:
PDB tahun 2014= US$272,02 miliar
PDB tahun 2013= US$250,18 miliar

Ditanya: Laju pertumbuhan ekonomi

Jawab:

[Rumus Laju Pertumbuhan Ekonomi]
Laju pertumbuhan ekonomi tahun t= PDBt - PDBt-1/ PDBt-1 × 100%

[Keterangan]
tahun t= tahun terhitung
PDBt= PDB tahun terhitung
PDBt-1= PDB sebelum tahun terhitung

Laju pertumbuhan ekonomi tahun 2014= PDB tahun 2014-PDB tahun 2013/ PDB tahun 2013 × 100%

Laju pertumbuhan ekonomi tahun 2014= US$272,02 miliar-US$250,18 miliar/US$272,02 miliar × 100%

Laju pertumbuhan ekonomi tahun 2014= US$21,84 miliar/272,02 miliar × 100%

Laju pertumbuhan ekonomi tahun 2014= 8,03%

Jadi, laju pertumbuhan ekonomi Filipina pada tahun 2014 adalah sebesar 8,03%.

[tex]=\frac{10}{640}[/tex]×100%

=1,5625%

36. pada februari 2014 A menabung di bank sebesar 2.000.000,00 dengan bunga 8% per tahun. Pada akhir agustus 2014 maka uang A akan menjadi...


Besar nya bunga dalam suatu pinjaman di nyatakan dalam % ( persen ) dan disebut dengan suku bunga. Suku bunga per tahun adalah perbandingan antara bunga yg diperoleh dengan modal awal per tahun.

Suku bunga = [tex] \frac {bunga yg diperoleh} {modal awal}[/tex] x 100%

Sedangkan untuk modal akhir menabung selama beberapa periode adalah:

Mn = Mo ( 1 + nb )

dimana :

Mn = modal akhir

Mo = modal awal  

n = jumlah periode  

b = suku bunga dalam % ( persen ) per periode

Pembahasan  

Mo = Rp 2.000.000

b = 8%

Dari Februari 2014 ke Agustus 2014 adalah selama 7 bulan

n = 7 bulan  

n = [tex] \frac {7} {12}[/tex] tahun

Besar uang A pada akhir Agustus 2014 adalah :

Mn = Mo ( 1 + nb )

Mn = Rp 2.000.000 ( 1 + [tex] \frac {7} {12}[/tex] x 8% )
