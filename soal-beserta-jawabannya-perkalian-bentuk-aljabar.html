coba dikoreksi ulang. semoga membantu1) 4x (5y^2 + 6y)
=> (4x)(5y^2) + (4x)(6y)
=> 20xy^2 + 24xy

2) (2x - 4) (12xy + 5y)
=> (2x)(12xy) + (2x)(5y) + (- 4)(12xy) + (- 4)(5y)
=> 24x^2y + 10xy + (- 48xy) + (- 20y)
