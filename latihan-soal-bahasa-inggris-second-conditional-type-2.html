22. kak bantu buatin 1 paragrph bahasa inggris di dalam nya ada kalimat conditional type 1


Sebelum membuat text dalam bahasa Inggris menggunakan conditional type 1, sebaiknya kita memahami dulu bentuk conditinal type 1.

Conditional Sentences type 1 adalah kalimat pengandaian. Kalimat pengandaian ini berdasarkan fakta yang ada dan digunakan untuk menyatakan situasi yang mungkin terjadi atau tidak mungkin terjadi.

Pada umumnya kalimat pengandaian (conditional sentences) terdiri dari dua bagian yaitu "main clause" dan "if clause." Jadi, pada kalimat conditional sentences type 1 ini,  kondisi pada "main clause" akan terjadi jika syarat -syarat pada "if clause" terpenuhi.

Secara penulisan tidak masalah apabila anda menuliskan "main clause" terlebih dahulu atau "if clause" terlebih dahulu.

Formula untuk Conditional Sentences Type 1 :

                     If + Subject + Verb (present), S + Will + Verb 1

                                              atau

                     Subject + Will + Verb 1, if +Subject

PEMBAHASAN

Contoh text menggunakan conditional type 1 :

Today, I have a lot of works to do. I try to manage to finish all of them. So, I think if I come home early I will be able to do my homework. I write down all the tasks that I have to do that day, if the task finish, I will mark them "done".

25. Apa contoh type 1 conditional dalam bahasa inggris


Jawaban:

Conditional Sentence type 1 digunakan ketika kita ingin mengungkapkan sesuatu yang memiliki kemungkin akan terjadi di masa yang akan datang.

If+simple present, simple future (will)

Contoh:

"If i come home early, i will get some rest"

"Jika aku pulang lebih awal, aku akan beristirahat"

Nah jadi maksudnya jika kamu pulang lebih awal, kemungkinan untuk bisa beristirahat itu ada dan bisa terjadi.

29. apa yang membedakan antara conditional type 2 dan conditional sentence type 3 ???


Conditional type 2 menggunakan rumus

If + S + V2, S + would + V1

Contohnya:

If I came, she would be mad.

Conditional type 3 menggunakan rumus

If + S + had + V3, S + would have + V3

Contohnya:

If hadn't been sick, I wouldn't have missed the lesson.
conditional type 3 utk kejadian masa lampau atau simple past tense, kejadian yang harus sudah dipenuhi di masa lalu. Rumus Conditional Sentence Type 3
If +  Subjek + Had + V3 +S+ M.2 + Have + V.3

conditional type 2 ini digunakan utk kejadian di masa sekarang atau present yang akan terjadi jika kondisi yang ada berbeda.
Rumus Conditional Sentence Type 2
If + Subjek +V.2/Were +S + M2 + V.1M2 (Modal bentuk 2) : Would,could

35. mereka akan membeli rumah, seandainya mereka mempunyai banyak uangtolong ubah ke bahasa inggris, setelah itu buat ke dalam conditional type 1 2 dan 3​


1. They will buy a house, if they have much money

Conditional sentence type 1 :

Rumus dari kalimat ini adalah: if + simple present, simple future “will” / imperative

Jadi, "If they have much money, they will buy a house"

Conditional sentence type 2 :

Dalam bentuk kalimat ini, rumus yang digunakan adalah if + simple past/were, would/could/might + bare infinitive

Jadi, "If they had much money, they would buy a house"

Conditional sentence type 3 :

Rumus yang digunakan adalah if + past perfect, would/should/could/might + have + past participle.

Jadi "If they had money, they would have bought a house"
