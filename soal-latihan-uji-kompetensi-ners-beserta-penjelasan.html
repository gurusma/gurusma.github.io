Jawaban beserta jalan penyelesaiannya untuk uji kompetensi 5. Kemungkinan yang dimaksud adalah Uji Kompetensi 5 hal 53 kelas VII yaitu kumpulan latihan soal tentang perbandingan. Berikut akan dibahas jawabannya satu per satu.

[tex]\frac{11}{42} = \frac{x}{100} [/tex]

9) Jamila adalah seorang perancang busana muda. Dia ingin membuka toko yang khusus menjual baju rancangannya di sebuah ruko. Dia menggambar rancangan toko seperti berikut.  Skala: ½ in = 3 meter. Lebar toko pada gambar adalah 2 in. Lebar toko sebenarnya yang  ingin dibuat Jamila adalah ... meter

Jawab

½ in ⇒ 3 m 2 in ⇒ x

Dengan perbandingan senilai, diperoleh

[tex]\frac{\frac{1}{2} \: in}{2 \: in} = \frac{3 \: m}{x} [/tex]

[tex]\frac{\frac{1}{2} \times 2}{2 \times 2} = \frac{3 \: m}{x} [/tex]

[tex]\frac{1}{4} = \frac{3 \: m}{x} [/tex]

x = 4 (3 m)

x = 12 m



15) Sebuah foto berukuran 3 cm × 4 cm. Apabila foto diperbesar dan sisi yang paling panjang menjadi 9 cm, maka panjang sisi terpendek menjadi ...

[tex]\frac{4}{3} = \frac{9 \: cm}{x} [/tex]

4x = 3(9 cm)

4x = 27 cm

16) Jika a : b = 3 : 4, maka (6a + b) : (4a + 5b) adalah ...

Jawab

a : b = 3 : 4

misal

a = 3x b = 4x

maka

(6a + b) : (4a + 5b)

= (6(3x) + 4x) : (4(3x) + 5(4x))

= (18x + 4x) : (12x + 20x)

= (22x) : (32x)

= 22 : 32

= 11 : 16



18) Jika (a + b) : (a – b) = 1 : 5, maka (a² – b²) : (a² + b²) sama dengan ....

Jawab

(a + b) : (a – b) = 1 : 5

[tex]\frac{a + b}{a - b} = \frac{1}{5}[/tex]

5(a + b) = 1(a – b)  

5a + 5b = a – b

5a – a = –b – 5b

4a = –6b

2a = –3b

a/b = –3/2

a : b = –3 : 2

misal

a = –3x b = 2x

maka

(a² – b²) : (a² + b²)

= ((–3x)² – (2x)²) : ((–3x)² + (2x)²)

= (9x² – 4x²) : (9x² + 4x²)

= 5x² : 13x²

= 5 : 13
